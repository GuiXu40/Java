#### :egg:一维数组
定义数组的两个步骤:声明数组+分配内存;结构如下:
```Java
数据类型 数组名称[]=null  //声明数组
数组名称=new 数据类型[长度];  //分配内存给数组
```
数组声明还有一种方式
```Java
数组类型[] 数组名称=null   //声明一维数组
```
数组声明后实际上是在栈内存中保存了数字的名称,接下来便是在堆内存中配置数组所需的内存.其中长度是告诉编译器,所声明的数组要存放多少个元素,而new是跟剧长度
在堆内存中开辟一块内存
```Java
int score[]=null
score=new int[3];
```
数组是引用类型的数据,因此数组变量score所保存的并非是数组的实体,而是数组堆内存的参考地址<br>
定义数组的简单书写方式
```java
数据类型 数组名[]=new 数据类型[个数]
```
#### :egg:数组中元素的表示方法
可以通过索引来访问数组中的元素(从0开始)
```Java
public class Main {
    public static void main(String[] args) {
	// write your code here
        int score[]=new int[3];    //定义数组
        System.out.println("score[0]:"+score[0]);
        System.out.println("score[1]:"+score[1]);
        System.out.println("score[2]:"+score[2]);
        for(int x=0;x<3;x++){
            System.out.println("score["+x+"]："+score[x]);
        }   //循环输出
    }
}
//score[0]:0
//score[1]:0
//score[2]:0
//score[0]：0
//score[1]：0
//score[2]：0
```
之所以全部是0,是因为声明的是整形数组,取默认值<br>
在Java中取得数组的长度(数组元素的长度)可以利用"数组名称.length"-->返回一个int型数据
```Java
public class Main {
    public static void main(String[] args) {
	// write your code here
        int score[]=new int[3];    //定义数组
        System.out.println("数组长度为:"+score.length);  //3
    }
}
```
#### :egg:数组的静态初始化
数组静态初始化:就是在数组声明的时候就初始化--**通过大括号**
```Java
数据类型 数组名[]={初值1,初值2..}
```
```Java
public class Main {
    public static void main(String[] args) {
	// write your code here
        int score[]={111,222,333};
        for(int i=0;i<3;i++){
            System.out.println("score["+i+"]:"+ score[i]);
        }
    }
}
//score[0]:111
//score[1]:222
//score[2]:333
```
#### :egg:数组应用范例
+ 计算数组中的最大值和最小值
```Java
public class Main {
    public static void main(String[] args) {
	// write your code here
        int score[]={67,89,87,69,90,100,75,90};
        int max=0;
        int min=0;
        max=min=score[0];
        for(int i=1;i<score.length;i++){
            if(score[i]>max){
                max=score[i];
            }
            if(score[i]<min){
                min=score[i];
            }
        }
        System.out.println("最大值："+max+",最小值："+min+"");
    }
}
//最大值：100,最小值：67
```
+ 冒泡排序
```Java
public class Main {
    public static void main(String[] args) {
	// write your code here
        int score[]={67,89,87,69,90,100,75,90};
        for(int i=1;i<score.length;i++){
            for(int j=0;j<score.length;j++){
                if(score[i]<score[j]){
                    int temp=score[j];
                    score[j]=score[i];
                    score[i]=temp;
                }
            }
        }
        for(int i=0;i<score.length;i++){
            System.out.println("score["+i+"]:"+ score[i]);
        }
    }
}
//score[0]:67
//score[1]:69
//score[2]:75
//score[3]:87
//score[4]:89
//score[5]:90
//score[6]:90
//score[7]:100
```
#### :egg:二维数组
二维数组的声明格式
```Java
int score[][];   //声明整形数组
score=new int[4][3];

///或者
int score[][]=new int[][];
```
声明二维数组必须告诉数组的行和列
```Java
public class Main {
    public static void main(String[] args) {
	// write your code here
        int score[][]=new int[4][3];
        score[1][0]=45;
        score[2][2]=89;
        score[3][1]=11;
        for(int i=0;i<score.length;i++){
            for (int j=0;j<score[i].length;j++){
                System.out.print(score[i][j]+"\t");
            }
            System.out.println("");
        }
    }
}
//0	0	0	
//45	0	0	
//0	0	89	
//0	11	0
```
未初始化的去默认值<br>
二维数组也可以利用大括号进行静态化初始
```Java
数据类型 数组名[][]={
    {第0行},
    {第1行},
    ...
    {第n行}
};
```
实例
```Java
public class Main {
    public static void main(String[] args) {
	// write your code here
        int score[][]={{11,22},{12,22,55},{0}};
        for(int i=0;i<score.length;i++){
            for (int j=0;j<score[i].length;j++){
                System.out.print(score[i][j]+"\t");
            }
            System.out.println("");
        }
    }
}
//11	22	
//12	22	55	
//0
```
